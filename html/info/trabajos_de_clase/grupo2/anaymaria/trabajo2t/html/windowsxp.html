Windows XP

Windows XP sale al mercado el 25 de octubre de 2001, con un diseño renovado que se centra en la facilidad y un centro de servicios de Ayuda y soporte técnico unificado. Es presentado n 25 idiomas. Desde mediados de los 70 hasta el lanzamiento de Windows XP, en el mundo se habían instalado unos 1.000.000 de ordenadores.

Windows XP se convertirá en unos de los productos de mayor venta en los próximos años, para beneficio de Microsoft. Se mejora la navegación en el menú Inicio, en la barra de tareas y en el Panel de control. Aumenta la preocupación por los virus y los piratas informáticos pero, a la vez disminuye estos problemas gracias a la posibilidad de realizar actualizaciones de seguridad online.

Caracterizado por un diseño visual limpio y simplificado, Windows XP Home Edition, facilita el acceso a las características más usadas. Windows XP, es diseñado para un uso en el hogar y ofrece mejoras como el Reproductor de Windows Media, el Asistente para configuración de red, Windows Movie Maker y funciones optimizadas para fotografía digital.

Windows XP Professional tiene características como un sistema de archivos cifrado y avanzadas funciones de red, restauración del sistema y la compatibilidad con Escritorio remoto.
