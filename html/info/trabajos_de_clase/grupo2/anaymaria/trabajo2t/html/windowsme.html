Windows Me

Diseñado para su uso en el hogar, Windows Me incorpora numerosas mejoras relacionadas con la reproducción de música, vídeo y redes domésticas. Además, supone un enorme avance en fiabilidad, comparándolo con versiones anteriores.

Algunos elementos que aparecen por primera vez en Windows Me, son: Restaurar sistema, Windows Movie Maker y las tecnologías del Reproductor de Microsoft Windows Media 7.
