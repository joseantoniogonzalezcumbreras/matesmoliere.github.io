Windows MS-DOS

En 1980 IBM contrató a Microsoft para escribir el sistema operativo del IBM PC, que saldría al mercado al año siguiente. Presionada por el poco tiempo disponible, Microsoft compró QDOS a Tim Paterson y le cambió el nombre por MS-DOS (Microsoft Disk Operating System). El contrato firmado con IBM permitía a Microsoft vender este sistema operativo a otras compañías, y en 1984 Microsoft había otorgado licencias de MS-DOS a 200 fabricantes de equipos informáticos. Así, este sistema operativo se convirtió en el más utilizado para PC, permitiendo a Microsoft crecer rápidamente en la década de los 80.

En febrero de 1981 se cargó por primera vez una versión del MS-DOS en una IBM que aún estando llena de errores, salió a la venta en julio. En 1982 salió al mercado Multiplan, un programa de hoja de cálculo, y el año siguiente se puso a la venta Microsoft Word. Microsoft fue una de las primeras compañías del sector que se dedicó desarrollar aplicaciones para Macintosh, una computadora personal creada por Apple Computer, obteniendo grandes éxitos de venta de programas para Macintosh como Word, Excel y Works.
