Windows 8

Windows 8 ofrece un sistema operativo enteramente renovado. Incluye consigo una interfaz totalmente nueva que trabaja perfectamente con funciones táctiles para el trabajo. La nueva barra de tareas y un sistema de administración optimizado son otras de las mejoras que incluye Windows 8.

La disposición de una pantalla de inicio con iconos en Windows 8 permite conectar con las personas, los archivos, las apps y los sitios web. El hecho de que las apps desempeñen un papel importantísimo permite que desde la pantalla Inicio puede accederse a un nuevo lugar donde conseguirlas: la Tienda Windows.

La interfaz de usuario de Windows 8 fue modificada para volverla más adecuada para su uso con pantallas táctiles, además de los tradicionales ratón y teclado.

Microsoft publica de la mano con Windows 8 a Windows RT. Este mismo se trabaja en algunas tabletas y ordenadores. Windows RT se diseñó para dispositivos ligeros y para una mayor duración de la batería. Trabaja solamente apps de la Tienda Windows.
