Aplicaciones para la Inteligencia Artificial

  1. Lingüística computacional: - Es un campo en el que se incluyen la informática y la lengua, que intenta crear de una forma lógica un lenguaje neutral desde el punto de vista computacional.

  2. Minería de datos (Data Mining): -Es el proceso de detecta la información accional de grandes conjuntos de datos. Utiliza análisis matemáticos para deducir patrones y tendencias que existen en los datos industriales.

  3. Medicina: -En la medicina se utiliza para sustituir funciones como la neuronal (en algunas partes) para mejorar la vida de las personas.

  4. Mundos virtuales: -Es la simulación de mundos o entornos físicos en un lugar virtual.

  5. Procesamiento de lenguaje natural (Natural Language Processing): -Es la disciplina encargada de producir sistemas que puedan hacer posible la comunicación, por medio del lenguaje verbal o textual.

  6. Robótica: Es una maquina o algo que sea multifuncional y reprogramable, diseñada para mover objetos o cosas, mediante movimientos programados.

  7. Mecatrónica: -La mecatrónica es un área multidisciplinar, ya que incorpora elementos de la electrónica, la mecánica, robótica, sistemas de computación y manufactura.

  8. Sistemas de apoyo a la decisión: -Es una herramienta enfocada al análisis de datos de una organización.

  9. Videojuegos: La inteligencia artificial ha entrado en el mundo de los videojuegos para aumentar la sensación de estar en la realidad en ellos, haciendo que personajes y jefes finales decidan funciones por si solo, aunque no esté muy desarrollado este campo dentro de los videojuegos, poco a poco van introduciéndolos y mejorándolo.

  10. Prototipos informáticos: Son las primeras pruebas antes de que el producto, máquina o software salga para su construcción.

  11. Análisis de sistemas dinámicos.

  12. Smart Process Management

  13. Simulación de multitudes

  14. Militar : Aquí la IA se utiliza para que las avanzadillas no sean personas, si no máquinas que puedan maniobrar solas para determinar la posición de un enemigo, además de bombardear o desmantelas bombas, aunque esta última no está muy desarrollada.
